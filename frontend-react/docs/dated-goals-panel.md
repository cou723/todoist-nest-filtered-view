# Dated Goals Panel 実装方針

## スコープ
- 「ゴールタスク」(@goal ラベル)のうち、期限が設定されている未完了タスクを一覧表示する。
- 表示項目はタスク名のみ。Todoist のタスク詳細ページへのアンカーとして扱う。
- ページング/最大件数/空状態メッセージはなし。空なら単に何も並ばない。

## 取得方針
- 取得クエリは Todoist フィルタで `@goal & !no date` を使用する。
  - Todoist ヘルプのフィルタ表に「See all tasks with a date assigned to them」 = `!no date` と明記されているため、期限付き抽出は `!no date` で足りる。
  - 完了済みタスクは Todoist Tasks API がそもそも返さないので追加条件は不要。
- ポートは既存の `TaskRepository.getAll(filter)` を流用する。UI/ユースケース側で新規フィルタ文字列を指定するだけでよい。

## 並び順
- 期限昇順で固定。
- 期限が同一の場合の優先順位は未指定。特に要件がなければ Todoist `child_order` など既存 `Task` の `order` をそのまま使う単純昇順を推奨（必要になれば別途指定する）。

## UI 要件
- パネルタイトルは「日付付きゴール」。`PanelWrapper` を利用し TaskListPanel と同等の枠で表示する。
- 行要素はタスク名のみをアンカー表示し、`https://todoist.com/showTask?id=<taskId>` に新規タブで遷移させる。
- 完了ボタンやメタ情報（ラベル/期日/優先度）は表示しない。

## 実装メモ
- ユースケース/フックは TaskListPanel と同等の構成にし、フィルタ文字列とソートロジックのみ差し替える形にすると影響範囲が小さい。
- タイムゾーン依存の表示要件はなし（期日表示自体をしないため）。
